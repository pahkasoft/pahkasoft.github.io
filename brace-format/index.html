About

BraceFormat is a TS/JS library for powerful string formatting using brace notation.

Note! Version 4 made BraceFormat and it's previous deprecated version StdFormat the same packag, only difference being global name when using iife browser bundle (window.BraceFormat vs. window.StdFormat).
